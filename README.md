# Employee Attrition Analysis Dashboard

An interactive Power BI dashboard analyzing employee attrition patterns across 1,470 records to help HR teams identify retention risk factors.

---

## 📊 Dashboard Overview

- **Total Employees**: 1,470 | **Attrition Rate**: 16.12% | **Employees Left**: 237
- Built with Power BI using DAX measures and interactive visualizations

---

## 🔍 Key Insights

### Top 5 High-Risk Roles
| Job Role | Employees Left |
|----------|---------------|
| Laboratory Technician | 62 |
| Sales Executive | 57 |
| Research Scientist | 47 |
| Sales Representative | 33 |
| Human Resources | 12 |

### Department Breakdown
- **Sales**: 133 leavers (56.12%)
- **R&D**: 92 leavers (38.82%)
- **HR**: 12 leavers (5.06%)

### Income Impact
- **Stayers**: $6,800 avg monthly
- **Leavers**: $4,800 avg monthly
- **Gap**: $2,000 (41% higher for stayers)

### Other Key Findings
- **Overtime workers**: 2.5x higher attrition
- **Frequent travelers**: 69 leavers (29.11%)
- **Job satisfaction (leavers)**: 2.47/4
- **Education fields**: Life Sciences (63), Medical (32) highest

---

## 🛠️ Features

- Interactive filters by department, job role, education
- KPI cards showing real-time metrics
- Visualizations: bar charts, pie charts, scatter plots
- DAX measures for custom calculations
- Drill-down capability

---

## 🚀 How to Use

1. Download `Employee_Attrition_Dashboard.pbix`
2. Open with Power BI Desktop
3. Use filters to explore different segments
4. Hover over charts for detailed tooltips
